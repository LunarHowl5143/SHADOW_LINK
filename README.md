# SHADOW LINK 2026 - Intelligence Portal

[cite_start]We are building the Karnataka State Police (KSP) Advanced Geospatial & Agentic AI Crime Analytics Platform[cite: 4]. [cite_start]This production-grade application transforms distributed record repositories into a proactive, state-level intelligence network[cite: 5].

## Core Architecture

* [cite_start]Frontend Tier: A highly responsive React single-page application bundled with Vite[cite: 7]. 
* [cite_start]Backend Tier: A pure Python serverless architecture utilizing a FastAPI framework wrapped natively for deployment inside Zoho Catalyst[cite: 10].
* [cite_start]Data Tier: A high-density relational schema built on the Zoho Catalyst Data Store[cite: 12].

## Initial Setup Instructions

Follow these steps to synchronize your local environment with the shadow network.

1. Clone the repository to your local machine.
2. Initialize the backend virtual environment:
   `python -m venv venv`
3. Activate the environment (Windows: `.\venv\Scripts\activate` | Mac/Linux: `source venv/bin/activate`).
4. Install the Python dependencies:
   `pip install -r functions/crime_api_handler/requirements.txt`
5. Navigate to the client directory and install frontend dependencies:
   `cd client`
   `npm install`
6. Authenticate with Zoho Catalyst CLI globally:
   `npm install -g zcatalyst-cli`
   `catalyst login`
